This tool runs only in your browser. The password is never transmitted, stored, or saved. Closing the page erases it. Still, avoid pasting a password you actively use on a machine you do not trust.
Entropy is estimated as length times the bits per character implied by the character types present. Crack time assumes a fast offline guessing rate and is a rough order-of-magnitude figure, not a guarantee.
